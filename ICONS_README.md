# Icon Files Required

The extension needs icon files for the Chrome extension. Create these PNG files:

## Required Icons

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Design Guidelines

### Colors
- **Primary**: UCLA Blue (#003DA5)
- **Accent**: UCLA Yellow (#FFD100)
