# 🎓 如何打开 MSBA AI Assistant

## 三种打开方式

### 方法1：快捷键（推荐）
- **Windows/Linux**: `Ctrl + Shift + B`
- **Mac**: `Command + Shift + B`

> ⚠️ 注意：已改为 `Ctrl+Shift+B` 避免与旧的 Kyle's AI Agent (`Ctrl+K`) 冲突

### 方法2：点击浮动按钮
- 在任何网页的**右下角**会有一个蓝色的 🎓 按钮
- 点击即可打开/关闭 MSBA AI Assistant

### 方法3：通过扩展图标
1. 点击 Chrome 工具栏中的扩展图标
2. 这会打开设置弹窗（不是 Assistant）
3. 设置完成后，使用快捷键或浮动按钮打开 Assistant

## 🔧 修改快捷键（可选）

如果你想使用 `Ctrl+K` 打开新的 MSBA Agent：

1. 在 Chrome 中访问 `chrome://extensions/shortcuts`
2. 找到 "UCLA MSBA AI Assistant"
3. 点击 "Toggle MSBA AI Assistant" 旁边的铅笔图标
4. 设置为你想要的快捷键（例如 `Ctrl+K`）

> ⚠️ 注意：如果旧 Agent 也在使用 `Ctrl+K`，会有冲突。建议：
> - 旧 Agent 使用 `Ctrl+K`
> - 新 MSBA Agent 使用 `Ctrl+Shift+B` 或 `Ctrl+Shift+M`

## ✅ 验证是否正常工作

1. 打开任意网页（例如 MSBA 官网）
2. 你应该看到右下角有一个蓝色的 🎓 按钮
3. 点击按钮或按 `Ctrl+Shift+B`
4. 应该会弹出 MSBA AI Assistant 侧边栏

## 🐛 如果看不到浮动按钮

1. 检查扩展是否已加载
2. 刷新网页（F5）
3. 检查浏览器控制台（F12）是否有错误
4. 确认 API key 已配置
